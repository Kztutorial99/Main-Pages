Tutorial Termux

Panduan lengkap instalasi, konfigurasi, perintah dasar, manajemen package, dan tips & trik Termux — 100% tanpa root di Android.

Cara benar menginstall Termux di Android tanpa root. Jangan install dari Play Store karena versinya sudah usang dan banyak tools yang tidak berjalan.

Wajib dibaca: Install Termux dari F-Droid atau GitHub Releases, BUKAN dari Google Play Store. Versi Play Store tidak diupdate sejak 2020 dan tidak kompatibel dengan banyak package modern.

Langkah Instalasi

1
Download Termux dari F-Droid
Buka browser di Android, download F-Droid app terlebih dahulu dari f-droid.org, lalu cari dan install Termux dari sana. Atau download APK langsung dari GitHub Releases Termux.
Link langsung: github.com/termux/termux-app/releases — download file termux-app_v*.apk
2
Izinkan Install dari Sumber Tidak Dikenal
Saat install APK, Android akan meminta izin "Install Unknown Apps". Aktifkan izin tersebut untuk browser atau file manager yang kamu gunakan.
3
Jalankan Termux & Update Pertama Kali
Buka Termux, lalu jalankan perintah berikut untuk update semua package ke versi terbaru:
# Update dan upgrade semua package pkg update && pkg upgrade -y
4
Izinkan Akses Storage
Agar Termux bisa mengakses file di penyimpanan internal Android (folder Download, dll), jalankan:
termux-setup-storage
Akan muncul popup izin akses storage — pilih Izinkan. Setelah itu folder ~/storage/ akan tersedia.
5
Install Tools Dasar
Install package penting yang akan sering digunakan:
# Install tools dasar esensial pkg install -y curl wget git python nodejs

Konfigurasi Termux dari awal agar lebih nyaman digunakan setiap hari — environment, shell, editor, dan storage.

Setup Terminal Lengkap (One Script)

Copy dan jalankan script ini untuk setup Termux secara otomatis — update, storage, tools dasar, dan konfigurasi.

#!/data/data/com.termux/files/usr/bin/bash # ===== TERMUX SETUP OTOMATIS ===== # Step 1: Update semua package echo "[1/5] Update packages..." pkg update && pkg upgrade -y # Step 2: Aktifkan akses storage echo "[2/5] Setup storage..." termux-setup-storage # Step 3: Install tools dasar echo "[3/5] Install tools dasar..." pkg install -y curl wget git python nodejs neovim htop tmux fzf zsh # Step 4: Install Python pip packages echo "[4/5] Install pip packages..." pip install requests httpx rich # Step 5: Set vim keybinding di bash echo "[5/5] Konfigurasi shell..." echo 'set -o vi' >> ~/.bashrc echo 'alias ll="ls -la"' >> ~/.bashrc echo 'alias cls="clear"' >> ~/.bashrc source ~/.bashrc echo "Setup selesai!"

Konfigurasi .bashrc / .zshrc

Tambah alias dan konfigurasi ini ke file ~/.bashrc atau ~/.zshrc agar berlaku setiap kali Termux dibuka.

# === ALIAS BERGUNA === alias ll='ls -la' alias la='ls -A' alias cls='clear' alias update='pkg update && pkg upgrade -y' alias myip='curl ifconfig.me' alias py='python3' # === PATH untuk Go binary === export PATH=$PATH:$HOME/go/bin # === Prompt warna === export PS1='\[\e[32m\]\u\[\e[0m\]@\[\e[34m\]\h\[\e[0m\]:\[\e[33m\]\w\[\e[0m\]\$ '

Tips, trik, dan hal penting yang perlu diketahui agar pengalaman menggunakan Termux lebih lancar dan aman.

Keamanan & Hal Penting

Jangan install dari Google Play Store
Versi Play Store tidak diupdate sejak 2020. Gunakan F-Droid atau GitHub Releases. Package di versi lama tidak kompatibel dengan repository Termux saat ini.
Selalu jalankan update sebelum install
Sebelum install package baru, jalankan pkg update && pkg upgrade -y agar repository dan package sudah yang terbaru. Ini menghindari konflik dependency.
Gunakan Tab untuk autocomplete
Tekan Tab saat mengetik perintah atau nama file untuk autocomplete otomatis. Tekan dua kali Tab Tab untuk melihat semua kemungkinan yang tersedia.
Gunakan tmux agar session tidak hilang
Jalankan proses panjang di dalam tmux. Kalau Termux tertutup atau layar mati, session tmux tetap berjalan di background. Sambung kembali dengan tmux attach.
Install tools security dengan benar
Tools seperti Nmap, Hydra, SQLmap hanya untuk penggunaan etis — uji pada sistemmu sendiri atau dengan izin pemilik. Semua tools di sini 100% berjalan tanpa root.
Pantau penggunaan storage
Termux menggunakan storage internal Android. Cek penggunaan dengan du -sh ~/* atau df -h. Bersihkan cache dengan pkg autoclean secara berkala.
